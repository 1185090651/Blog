## 博客

这个项目大家应该都比较熟悉，一般学习一种语言或者框架都会拿一个具有简单 CRUD 的项目练手，博客就是其中一个。

这个项目也不例外。但我更希望它能够脱离复杂业务只从技术角度出发，回顾与总结我学过的技术。它更像一只小白鼠🐭，我可以把所学的技术全实验它身上。

废话不多说，Let's Do it !
